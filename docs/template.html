Monexa Flow TRIAL

Sistema de Auditoria Bancaria Profesional para Itau Uruguay

1. Escaner Automatico de Movimientos

El corazon de Monexa. Se activa automaticamente al ingresar a la pagina del banco Itau y escanea todas las filas de la tabla de movimientos inyectando columnas de auditoria.

  • Detecta automaticamente cada movimiento (fecha, concepto, debito, credito, saldo)
  • Inyecta columnas extras: Etiqueta, Nota y Estado (semaforo visual)
  • Resalta cada fila con color de fondo segun su estado de revision
  • Detecta el numero de cuenta activa y lo muestra en un chip sobre el boton flotante
  • Usa un MutationObserver para re-escanear cuando el banco carga nuevos datos
  • Filtra cotizaciones de divisas (U$S, Pizarra, Brou)

2. Sistema de Estados (Semaforo)

Cada movimiento puede ciclar entre 4 estados haciendo click:

EstadoColorIconoSignificado
Sin RevisarGris○No fue tocado aun
ValidadoVerde✓Aprobado por el auditor
ObservadoAmbar!Requiere atencion
RechazadoRojo×Irregular

Ciclo: Sin Revisar → Validado → Observado → Rechazado → Sin Revisar

3. Motor de Reglas (Auto-Match)

Permite definir reglas de texto para que los movimientos se clasifiquen automaticamente.

  • Patron + Etiqueta: Si el concepto contiene el patron, se asigna la etiqueta y se marca Validado
  • Limpieza inteligente: Elimina caracteres invisibles y dobles espacios
  • Aplicacion retroactiva: Las reglas nuevas se aplican a registros pasados sin etiquetar
  • Exportar / Importar: Las reglas se exportan e importan entre instalaciones

Ejemplo: COMPRA → Gastos | CRE. CAMBIOSOP → Alquileres

4. Buscador en Tiempo Real

Motor de busqueda que filtra por concepto, nota o etiqueta en todas las transacciones. Resultados al instante.

5. Panel de Control Lateral

Se abre al hacer click en el boton flotante MX (naranja, esquina inferior derecha).

  • Header: MONEXA FLOW TRIAL con gradiente naranja
  • Sesion: Nombre del auditor + Cerrar Sesion
  • Estadisticas: 3 cuadros (Validados / Observados / Rechazados)
  • Motor de Reglas: Formulario para crear y gestionar reglas
  • Buscador: Campo de busqueda de transacciones
  • Acciones: Exportar Excel, Exportar Reglas, Importar, Dashboard
