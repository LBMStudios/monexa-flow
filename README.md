# Monexa Flow 🚀

Monexa Flow es una extensión de Chrome profesional diseñada para la **automatización y auditoría de movimientos bancarios** (Itaú Uruguay). Permite capturar transacciones directamente desde el portal bancario y organizarlas en un dashboard premium de consulta local.

## 📂 Estructura del Proyecto

-   **/extension**: Contiene el código fuente de la extensión de Chrome.
-   **/landing**: Página promocional y sistema de descarga directa.

## 🛠️ Instalación (Sideloading)

Como esta es una herramienta de auditoría avanzada, puedes instalarla directamente siguiendo estos pasos:

1.  **Descarga** el archivo `.zip` desde la [Landing Page](https://lbmstudios.github.com/monexa-flow/landing/index.html) o directamente desde este repositorio.
2.  **Descomprime** el contenido en una carpeta local.
3.  Abre Google Chrome y ve a `chrome://extensions`.
4.  Activa el **"Modo de desarrollador"** (arriba a la derecha).
5.  Haz clic en **"Cargar extensión sin empaquetar"** y selecciona la carpeta `/extension`.

## ✨ Características Principales

-   **Escaneo Pasivo:** Captura movimientos mientras navegas, sin interceptar credenciales.
-   **Dashboard Premium:** Análisis visual de gastos, ingresos y filtrado avanzado.
-   **Privacidad Total:** Los datos se guardan exclusivamente en el almacenamiento local de tu navegador.
-   **Sistema de Avisos:** Notificaciones automáticas cuando hay una nueva versión disponible.
